# Images Directory

Please add the following images to this directory:

## Required Images:

1. **logo.png** - Main logo for the header (recommended: 40x40px or larger)
2. **instructions-icon.png** - Icon for step 1 header (recommended: 80x80px)
3. **user-icon.png** - Icon for step 2 header (recommended: 80x80px)
4. **services-icon.png** - Icon for step 3 header (recommended: 80x80px)

## Instruction Images (Step 1):
- **instruction-1.png** - Prepare environment image (120x120px, will be displayed as circular, no frame)
- **instruction-2.png** - Distance image (120x120px, will be displayed as circular, no frame)
- **instruction-3.png** - Focus image (120x120px, will be displayed as circular, no frame)

**Note:** These images will be automatically cropped to a circle shape. Make sure your images are square (120x120px) for best results.

## Form Icons (Step 2):
- **name-icon.png** - Name input icon (recommended: 24x24px)
- **age-icon.png** - Age input icon (recommended: 24x24px)

## Option Images (Step 3):
- **vision-test.png** - Vision test option image (recommended: 100x100px)
- **lazy-eye.png** - Lazy eye option image (recommended: 100x100px)
- **guidelines.png** - Guidelines option image (recommended: 100x100px)
- **report.png** - Report option image (recommended: 100x100px)

## Notes:
- All images should be in PNG format with transparent backgrounds (recommended)
- Images will automatically hide if they don't exist (using onerror handler)
- Recommended format: PNG with transparent background
- Recommended size: As specified above for optimal display
